# consumerSpring
Конвейер для приема сообщений из Kafka и их постоянного хранения в реляционной базе данных на фрейморке SPRING.
1. Подписывается на топик "postedmessages" и обрабатывает поступающие сообщения в реальном времени.
2. Полученное сообщение сохраняется в таблицу "messages".
3. База данных назначает уникальные идентификаторы сохраненным сообщениям.
4. Выводит в консоль информацию о полученных сообщениях и результатах сохранения с присвоенными ID.
